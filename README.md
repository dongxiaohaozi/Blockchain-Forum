
# Installation

## Spring Boot

1. Change directory to ./backend

```sh
cd ./backend
```

2. Copy .env file and set the value

```sh
cp .env.example .env
```

3. Install maven dependencies

```sh
mvn install
```

4. Run on localhost:8080

```sh
mvn spring-boot:run
```

## React.js

1. Change directory to ./frontend

```sh
cd ./frontend
```

2. Install npm packages

```sh
npm install
```

3. Run on localhost:3000

```sh
npm start
```
